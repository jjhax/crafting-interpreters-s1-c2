# Challenge 2

This is the second challenge from section one of the book: *Crafting Interpreters*::wq

> Get a “Hello, world!” program written and running in Java. Set up whatever makefiles or IDE projects you need to get it working. If you have a debugger, get comfortable with it and step through your program as it runs.

## Running

```bash
make
make jar
make run
```

## Debugging

Use the Java Debugger `jdb` from the command line to debug. Read the documentation.

A cheatsheet can be found here: https://gist.github.com/ranjitiyer/a056da16a7da95a3e4aba96bc2a73404
